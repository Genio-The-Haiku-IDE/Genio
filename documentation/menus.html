The Menus

The vast majority of Genio's menu items are trivial or obvious.
Let's go through all menus and only pick those items that deserve a bit of explanation.

The Genio menu

Genio project… opens the GitHub page of the Genio project, as shortcut to file an issue or participate in discussions.

Settings… opens the application's Settings window.

The File menu

New offers a sub-menu with template files and folders.
In it is an item Edit user templates… that opens a folder where you can put your own files/folders to use as templates. Those, too, will then be available in the New sub-menu.

When you're editing an .rdef resource, you can use Import as RDEF array… to choose any file that'll then get inserted as array at the end of the .rdef file.

Show in projects browser will scroll to and select the file you're currently editing in the Projects pane. It's also available from the context menu of a file's tab above the editor view.

The Edit menu

Trim trailing whitespace removes all spaces and tabs at the end of the line for the whole file.

Autocomplete is described on the Editor page under Code completion.

Line endings converts the end-of-line character of the current file to either LF (Haiku, Unix, macOS), CRLF (Windows, Dos) or CR (Classic Mac OS).

Language lets you set the programming language of the current file manually. Genio normally auto-detects this correctly.

The View menu

You can Zoom in/out, ALT + -, the editor view, which can be handy to scroll through long source files. Perhaps more conveniently, this can also be done by scrolling up/down with the mouse wheel while holding ALT.

The Search menu

Find, ALT F, and Replace, ALT R, are described on the Toolbar page under Find bar and Replace bar.

The Project menu

Open project… and Open remote project… are described on the QuickStart page under Local project and Remote project.

Build project and Clean project will build or clean the active project according to the set Build mode, either as "release" or "debug".

Run target will launch the built binary.

For the above to work, you need to have your project properly configured in the Project settings….

If you're building a localized application, you can generate new en.catkeys with Make catkeys and do a Make bindcatalogs to add the available translations as resources to the built binary.

If you're using EditorConfig to manage your coding style settings, you can choose Reload .editorconfig to force an update of the open files when you've edited an .editorconfig file.

The Git menu

This menu offers a number of often used Git commands. It's only available if you're working with a git repository, of course. Check the Panes page under Source control how to initiate a git repo.

All git commands print their output in the Console I/O tab.

The Window menu

You can show/hide panes and toolbar under Appearance.

Genio's window can be made Fullscreen (ALT ENTER).

In Focus mode (SHIFT ALT ENTER) the toolbar and the projects and output panes are additionally hidden to minimize distractions.

Hit ALT ENTER again to return to normal mode.
