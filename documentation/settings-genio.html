The Genio Settings

From the list on the left, you choose the different settings categories.

Defaults will reset all settings to their defaults.

General

Projects folder This sets the default location for newly created projects.
Auto collapse/expand projects Genio will automatically expand a project you set to "active" and collapse the other currently open projects.
Show full path in window title Shows the full path or just the name of the current file.
Log destination Sets where Genio's log output gets written: stdout, stderr, syslog or BeDC (see here).
Log level Sets the verbosity of Genio's log output. After Off it gets increasingly detailed: Error, Info, Debug and Trace.

General / Startup

These settings determine how Genio will behave on startup. You decide if the projects and files that were open when you last quit Genio are reloaded when it launches.

General / Appearance

These settings determine Genio's default look. Besides setting which panes and bars are shown by default when Genio launches, you can also opt to Use smaller icons in toolbar to save precious space.

Editor

Font
Font size		 You can set a font family and size if the system defaults don't work for you.
Enable syntax highlighting
Enables brace matching		 Colors keywords etc. and highlights opening/closing braces.
Save caret position Genio will remember the cursor position when you close and re-open a file.
Ignore .editorconfig Opt out of code formatting by EditorConfig (if your project uses that).

The rest of the settings are only applied if no .editorconfig is being used. That is, you either have the above Ignore .editorconfig checked, or your project simply doesn't use EditorConfig.

Trim trailing whitespace on save Automatically removes those pesky spaces and tabs at the end of lines when you save a file.
Convert tabs to spaces While active, entered tabs are replaced with spaces. Tabs that are already in a file are not replaced.
Tab width Sets the width of tabs in spaces.

Editor / Visual

Editor style Choose from a number of different color themes for the editor view.
Show line number
Show comment margin
Show folding margin		 Toggles the margins on the left (see Elements of the editor view).
Mark caret line Highlights the current line of the cursor.
Show whitespace Shows symbols for spaces and tabs. This can also be toggled from the View menu or the toolbar.
Show line endings Shows symbols for line breaks – LF, CRLF, CR, depending on the setting of the file. This can also be toggled from the View menu.
Wrap lines Wraps lines that are longer than the editor view's width. This can also be toggled from the View menu or the toolbar.
Show vertical ruler Draws a vertical line at the column you set with the spinner below. It helps you keeping a limit on characters per line.
Editor zoom Sets the default zoom of the editor view. It can easily be overridden with the commands in the View menu or by ALT + mousewheeling.

Editor / Find

The checkboxes set the defaults for The Find bar.

Wrap around Continue the search at the beginning of the file when it has reached the end (and vice-versa, if you reverse the direction by jumping to the previous match).
Whole word Does not find partial matches.
Match case Respect upper and lower-case.
Exclude folders A comma separated list of folders that will be ignored when searching. Use '*' as wildcard.

Build

Wrap lines in console Wraps lines in the Console I/O tab of the output pane.
Console banner Prints the executed command at the start and end of the build process in the Console I/O tab of the output pane.
Auto-Build on resource save Builds the project automatically if changes to any of its files are saved.
Auto-Save changed files when building Saves all changed files automatically when a build was started.

LSP

LSP is Clang's language server protocol. Genio uses it for autocompletion, signature help, jumping to definition / implementation / declaration, quick fix, etc. Please also have a look at Configuring clang's LSP.

Log level Sets the verbosity of LSP's log output. It gets increasingly detailed: Error, Info and Trace.

You'll have to restart Genio for changes to take effect.

Source control

Show repository outline Shows the branches of the project's repository in a hierarchical structure.
